AI-Driven Predictive Maintenance for Renewable Energy Assets

Using Web-Sourced Sensor Data

About

This project demonstrates AI-driven predictive maintenance using OpenWeather and NASA POWER API data for renewable energy assets.

More About This Project

Our system integrates satellite-based reanalysis data with real-time forecasts to provide insights into wind speed, temperature, humidity, and solar radiation. By combining multiple data sources, the dashboard supports predictive maintenance strategies for renewable energy assets, ensuring reliability and efficiency.
